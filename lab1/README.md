# /lab1 - реализация классификатора

Реализация классификатора на основе методов парзеновского окна и kNN с различными ядрами и метриками.

## Структура

- `exp_note/` - Jupyter notebook с графиками и визуализацией
- `main.py` - точка входа для модульной версии
- `kernels/` - реализация ядерных функций
- `metrics/` - функции расстояний и метрики близости
- `scores/` - метрики качества классификации
- `knn_parzen.py` - основной класс классификатора
- `model_selection/` - утилиты для валидации (LOO, cross_val_score, train_test_split)

## Особенности структуры

Код представлен в двух вариантах по историческим причинам:
1. **Модульная структура** (основные папки + `main.py`) — чистый код с разделением ответственности
2. **Jupyter notebook** (`exp_note/`) — для экспериментов и визуализации
